# Screenshots

Add screenshots of the running app here so they render in the main README.

Capture these from the live app (sign in with the demo button first), then save them with these exact names:

- `landing.png` — the landing page (`/`)
- `dashboard.png` — the dashboard (`/dashboard`)
- `comparison.png` — quotation comparison (`/quotations/:id` → Compare)
- `reports.png` — reports & analytics (`/reports`)
- `invoice.png` — an invoice document (`/invoices/:id` → View)

Tip: use full-page screenshots at ~1440px width for a clean look.
